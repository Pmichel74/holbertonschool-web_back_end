## ğŸš€ Getting Started

### Prerequisites

Before starting, ensure you have the following installed:

- ![Node.js](https://img.shields.io/badge/Node.js-14+-green) **Node.js** version 14 or higher
- ![Python](https://img.shields.io/badge/Python-3.7+-blue) **Python** version 3.7 or higher
- ![MongoDB](https://img.shields.io/badge/MongoDB-4.4+-green) **MongoDB** version 4.4 or higher
- ![npm](https://img.shields.io/badge/npm-6+-red) **npm** version 6 or higher

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/holberton/holbertonschool-web_back_end.git
   cd holbertonschool-web_back_end
   ```

2. **Install Node.js dependencies**
   ```bash
   npm install
   ```

3. **Set up Python environment** (recommended)
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

4. **Install Python dependencies** (if any)
   ```bash
   pip install -r requirements.txt  # If available in specific modules
   ```

5. **Verify installation**
   ```bash
   # Test JavaScript setup
   npm run lint
   npm test
   
   # Test Python setup
   python3 --version
   
   # Test MongoDB connection
   mongosh --eval "db.runCommand('ping')"
   ```
